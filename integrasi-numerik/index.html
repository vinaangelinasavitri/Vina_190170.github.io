    INTEGRASI NUMERIK¶

    Metode integrasi numerik adalah suatu cara untuk menghitung aproksima luas daerah di bawah fungsi yang dimaksud pada selang yang diberikan. Bentuk dasar integral adalah : $$ I = \int ^{b}_{a}f\left( x\right) dx $$ Dapat diartikan bahwa integral dari fungsi f(x) terhadap peubah bebas x yang dievaluasi mulai dari x = a hingga x = b. Pendekatan terhadap integral diatas dapat dinyatakan sebagai berikut :

    \[ I\left( x\right) = \sum ^{N}_{i=1}w_{i}y\left( x_{1}\right) \]

    Klasifikasi Metode Integrasi Numerik¶

    Metode Newton-Cotes¶

    Metode integrasi Newton-Cotes secara umum merupakan metode integrasi yang dilakukan dengan membagi area di bawah kurva suatu fungsi menjadi beberapa panel dengan terlebih dahulu menetapkan batas atas dan batas bawah interval.

    \[ I=\int ^{b}_{a}f\left( x\right)dx\approx\int ^{b}_{a}p_{n}\left( x\right)dx \]

    Nilai hampiran f(x) dengan polinomial

    \[ P_{n}\left( x\right) =a_{0} + a_{1}x + a_{2}x^2 + ... + a_{n-1}x^{n-1} + a_{n}x^{n} \]

    Luas daerah yang diarsir L dapat dihitung dengan :

    \[ L = \int ^{b}_{a}f\left( x\right)dx \]

    • Kaidah Trapesium (Trapezoidal rule)¶

    Pendekatan trapezoida dilakukan dengan melakukan pendekatan area di bawah kurva fungsi y = f(x) dengan subinterval [ x (i), x (i+1) ] menggunakan trapesium. Berikut fungsi persamaan yang merupakan proses integrasi menggunakan metode trapezoida :

    $$ \int ^{b} _{a} f \left ( x\right)dx = \lim _{n\rightarrow \infty } \sum ^{n} _{i=1 }\dfrac{\left( C _{i+1}-C _{i}\right) \times \left( f\left ( c _{i+1}\right) + f\left( c _{i} \right) \right) } {m} $$

    dimana $$ c_{i} = a+\dfrac{\left( b-a\right) }{n}i $$

    Error dari metode ini dapat diestimasi menggunakan persamaan berikut :

    $$ \int ^ {b} _{a}h\left( x\right) dx = - \dfrac{\left( b-a\right) ^{3}}{12m ^{2}} f ^{\left( 2\right) }\left( \xi \right) $$

    Algoritma Metode Trapezoida :

    • Definisikan y = f(x)

    • Tentukan batas bawah (a) dan bawar atas integrasi (b)

    • Tentukan jumlah pembagi = n

    • Hitung h = (b-a)/n

    • Hitung

      \[ L = \dfrac{h}{2}\left( f_{0}+2\sum ^{n-1}_{i=1}f_{i}+f_{n}\right) \]

    Berdasarkan persamaan diatas, kita dapat melakukan perhitungan integral trapezoida dengan bahasa python dengan sintaks seperti berikut :

    def trapezoid(f,a,b,n):
    h = (b-a)/n
    sum= 0.0
    for i in range(1,n):
        x = a+1*h
        sum = sum+f(x)
    integral = (h/2)* (f(a)+2*sum+f(b))
    return integral

    • Metode Simpson ⅓¶

    Metode Simpson membagi subinterval [a,b] menjadi n subinterval, dimana n merupakan bilangan genap. Untuk setiap pasang subinterval, luas area di bawah fungsi f(x) ditaksir menggunakan polinomial derajat 2.

    $$ \int ^{b} _{a}f\left( x\right)dx \approx\int ^{b} _{a}\left( a _{0}+a _{1}x+a _{2}x ^{2}\right) dx $$

    Aturan komposisi simpson

    $$ h = \dfrac{\left( b-a\right) }{n} $$

    Dengan menggunakan aturan simpson, luas dari daerah yang dibatasi fungsi y = f(x) dan sumbu X dapat dihitung sebagai berikut :

    $$ L=\dfrac{h}{3}\left( f _{0}+4\sum _{i ganjil}f _{i}+2\sum _{i genap}f _{i}+f _{n}\right) $$

    Algoritma Metode Simpson ⅓

    • Tentukan fungsi f(x) dan selan integrasinya [a,b].
    • Tentukan jumlah subinterval n.
    • Hitung nilai selang subinterval h.
    • Tentukan awal integrasi dan akhir dengan hitung nilai f(a) dan f(b).
    • Untuk x = 1,2, ... , n-1 , jika ganjil, hitung : 4 x f(x), dan jika genap, hitung : 2 x f(x).
    • Jumlahkan nilai-nilai taksiran menggunakan persamaan diatas.

    Code Metode Simpson ⅓

    def simpson(f,a,b,n):
    h = (b-a)/n
    sum1= 0.0
    sum2= 0.0

    for i in range(1,n,2):
        x = a+i*h
        sum1=sum1+f(x)
    for i in range (2,n-1,2):
        x = a+1*h
        sum2=sum2+f(x)

    integral = (h/3)*(f(a) + 4*sum1 + 2*sum2 + f(b))

    return integral

    • Metode Simpson ⅜¶

    Pada Metode simpson ⅜ digunakan pendekatan polinomial derajat 3 untuk memperoleh hasil yang lebih baik. Bentuk umum integrasi metode simpson ⅜ :

    \[ L=\dfrac{3h} {8}\left( f _{0}+3\sum ^{n-1} _{i=1,i\neq3,6,9,...}f _{i}+2\sum ^{n-3} _{i=3,6,9,...}f _{i}+f _{n}\right) \]

    Algoritma Metode simpson ⅜

    • Tentukan fungsi f(x) dan selang integrasinya [a,b]
    • Tentukan jumlah subinterval n
    • Hitung nilai selang subinterval h
    • Tentukan awal integrasi dan akhir integrasi lalu hitung f(a) dan f(b)
    • Untuk x = 1,2, ... , n-1, jika bukan kelipatan 3, hitung : 3 x f(x) , dan juka kelipatan 3, hitung : 2 x f(x)
    • Jumlahkan nilai-nilai taksiran tersebut.

    Code Metode Simpson ⅜

      def simpson38(f,a,b,n):
    h = (b-a)/n
      sum1= 0.0
    sum2= 0.0

    for i in range(1,n,3):
          x = a+i*h
        sum1=sum1+f(x)
      for i in range (3,n-1,3):
        x = a+1*h
          sum2=sum2+f(x)

      integral = (3*h/8)*(f(a) + 3*sum1 + 2*sum2 + f(b))

      return integral

    Metode Romberg¶

    Metode Romberg adalah perluasan yang relatif mudah dari algoritma Newton-Cotes. Metode Romberg bekerja dengan menggunakan iterasi yang disempurnakan dari beberapa metode Newton-Cotes. Secara umum :

    \[ I_{j,k} = \dfrac{4^{k}I_{j,k-1}-I_{j-1,k-1}}{4^{r}-1} \]

    ​ Algoritma Metode Romberg

    • Tentukan fungsi f(x) dan selang integrasinya [a,b]
    • Tentukan jumlah subinterval
    • Bentuk matrik I dengan ukuran j x k yang akan menampung hasil perhitungan.
    • Untuk I(1,1) hitung integral fungsi menggunakan metode trapezoida.
    • Solusi integrasi diperoleh pada I(j,k).

    ​ Code Metode Romberg

    import math
import numpy as np

f = lambda x: math.exp(x)
a = 0, b = 1

def Trap(n,a,b):
    h = (b-a)/n
    sum = 0.0
    for i in range (1,n):
        sum = sum + 2*f(a+i*h)
    Trap = (h/2) * (f(a) + sum + f(b))

I = np.zeros((10,10))
ea = 100; n = 1; i = 1
I[1,1] = Trap(n,a,b)

print('Application \t ea \t\t et \t\t Integral')

while ea > 0.001:
    n = 2 ** i

    I[i+1 , 1] = Trap(n,a,b)

    #Romberg Integration
    for k in range (2, i+2):
        j = 2 + i - k
        I[j,k] = (4**(k-1)* I[j+1, k-1] - I[j, k-1])/(4**(k-1)-1)

    ea = abs((I[1, i+1] - I[2,i])/ I[1, i+1])*100
    et = abs((1.71828 - I[1,i+1])/1.7182)*100

    print('%f \t %f \t %f \t %f' %(i,ea,et,I[1,i+1]))

    i+=1
